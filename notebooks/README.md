# Notebooks

## 建議順序

1. `00_environment_check.ipynb`：確認 Python、Jupyter、ISLP 與常用套件可正常使用。
2. `01_practice_template.ipynb`：每次實作可複製此模板，依課堂任務填入。
3. `official/`：執行 `scripts/fetch_islp_labs.ps1` 後產生，放官方 ISLP labs。

## 命名規則

學生作業建議命名：

```text
HW_ch03_linear_regression_姓名.ipynb
HW_ch05_resampling_姓名.ipynb
```

## 提交前檢查

- 從 notebook 第一格重新執行到最後一格。
- 確認沒有私有路徑，例如 `C:\Users\...`。
- 圖表要有標題或簡短說明。
- 結論要包含限制。
