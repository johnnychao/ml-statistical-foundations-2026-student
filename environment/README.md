# Environment

## Lite environment

適合前幾週與多數表格式資料實作：

```powershell
python -m pip install -r environment\requirements-lite.txt
```

## Full ISLP labs environment

適合完整執行官方 labs，版本釘在 ISLP_labs `v2.2.3`：

```powershell
python -m pip install -r environment\requirements.txt
```

完整環境可能包含較重的 deep learning 相關套件，安裝時間會比較長。
