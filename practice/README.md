# 自主練習

本課程不要求學生繳交任何成果或資料。此資料夾提供自主練習方向，讓學生在課堂或課後自行實作、檢查理解。

## 練習檔命名建議

若要保存自己的練習 notebook，建議命名：

```text
practice_chXX_topic.ipynb
```

例如：

```text
practice_ch03_linear_regression.ipynb
practice_ch05_resampling.ipynb
```

## 自我檢查

- Notebook 可以從第一格執行到最後一格。
- 程式碼區塊能看懂每一步在做什麼。
- 圖表有標題或簡短解釋。
- 結論包含模型限制與下一步。
- 不把個資、私有資料或本機絕對路徑寫進 notebook。

## 建議練習重點

每次實作可以練習回答：

- 這次任務是 prediction、interpretation 還是 exploration？
- 目標變數是什麼？如果沒有目標變數，想探索什麼結構？
- 資料有哪些欄位、缺失值或明顯異常？
- baseline 模型結果如何？
- 模型限制是什麼？哪些結論不能解釋成因果？
