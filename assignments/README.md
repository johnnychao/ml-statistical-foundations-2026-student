# Assignments

## 作業提交格式

每次作業請提交一份 notebook，建議命名：

```text
HW_chXX_topic_姓名.ipynb
```

## 基本要求

- Notebook 可以從第一格執行到最後一格。
- 程式碼區塊要保留必要註解。
- 圖表要有標題或簡短解釋。
- 結論需包含模型限制與下一步。
- 不提交個資、私有資料或本機絕對路徑。

## 評分建議

| 項目 | 比重 | 說明 |
| --- | ---: | --- |
| 問題定義 | 20% | 能說明任務類型、目標變數與評估方式 |
| 資料檢查 | 20% | 有基本欄位、缺失值、分布或圖表檢查 |
| 模型流程 | 30% | 有合理 baseline、訓練與驗證流程 |
| 結果解釋 | 20% | 能用統計學習語言解釋結果與限制 |
| 可重現性 | 10% | notebook 可重新執行且無私有路徑 |
